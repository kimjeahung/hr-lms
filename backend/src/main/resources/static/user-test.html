LMS User API 테스트 콘솔

course/enrollment API를 브라우저에서 바로 테스트합니다. 기본 URL은 현재 접속 주소입니다.

1) 로그인/JWT

2) 사용자 수강 API (/api/user/enrollments)

버튼별 필수 파라미터
  • 신청 POST: userId, roundId
  • history/all/ongoing GET: userId
  • 상세/완료/일정상세 GET: enrollmentId
  • 진행률 PUT: enrollmentId, progress(0~100)
결과가 여기 표시됩니다.

3) 관리자 Course/Enrollment API

버튼별 필수 파라미터
  • 수강 등록 POST: userId, roundId
  • 수강 진행률 PATCH: enrollmentId, progress
  • 수강 상태 PUT: enrollmentId, status(NOT_STARTED/IN_PROGRESS/DONE)
  • 수강 일정 PUT: roundId, endDate(yyyy-MM-dd)
  • 강좌 생성/수정: title, description, category, targetRole, durationMin, thumbnailUrl
